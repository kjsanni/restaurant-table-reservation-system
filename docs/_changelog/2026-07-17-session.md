**Root Cause:** `getCustomerReservationHistory` and `getCustomerStats` functions were not exported from `reservation.dao.js`. **Fix:** Added missing exports. Removed `expectedTotal` from `getCustomerStats` GROUP BY query. Added `required: false` to Table association. Added `foreignKey` specification to `Reservation.hasMany(models.table)`.  
**Verification:** Customer profile endpoint returns 200 with valid data.
