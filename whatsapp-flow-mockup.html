Key Components

💬
WhatsApp Business API
Primary customer touchpoint — all interactions happen here
📍
Location Messages
Native WhatsApp location sharing with lat/lng extraction
🗺️
Geocoding (Nominatim)
Reverse geocode lat/lng → address, or forward geocode text → coords
💳
Paystack Payments
GHS via Mobile Money (MTN, Vodafone, AirtelTigo), cards, banks
🚚
ShaQ Express
Ghanaian delivery partner — creates packages, returns tracking
🗄️
Redis Sessions
24h TTL conversation state — cart, reservation, delivery progress
📅
NL Date/Time Parser
Understands "tomorrow", "7pm", "next Monday" in conversation
🇬🇭
Ghana Region Resolution
16 region bounding-box lookup from coordinates
